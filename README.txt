# ParanoiaPlayerOrganizer
A little window that shows all the stats of one Paranoia Character. Character stats can be changed by editing the text boxes or using the buttons

Example character string:

Buttons:
+     plus one xp point
++    plus ten xp points
+++   plus 100 xp points
-     minus one xp point
--    minus ten xp points
---   minus 100 xp points
^     promote one color rank
v     demote one color rank
:-)   heal one damage
:-(   deal one damage
:) :) add one additional clone
*     add one treason star
?     undefined
!     kill current clone
load  take character string and fill the text boxes
save  empty the textboxes and make a character string
